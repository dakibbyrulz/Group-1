# Group-1
This is a repository for group 1 of a 2026 Spring Semester MRE 320 class.


# explanation and content
 During this project, we wired up and tested 3 different sensors, a temp and humidity sensor, an ultrasonic sensor, and an infrared sensor.
 we took these findings and compared them to charts we found that contained what certain tolerances should be.
 in the files, you can find the wiring diagrams for each sensor as well as the results from testing.
