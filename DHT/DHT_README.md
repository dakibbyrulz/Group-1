# DHT Sensor

The DHT11 measures humidity and temperature, and is known for having relatively accurate readings for its size and how low-priced it is on the market, with a price range of $0.43-$6. The sensor reads temperature via a thermistor, and humidity via a capacitive humidity sensor. It has a reset period of 2 seconds

# Testing Plans and Overall Results

Testing for both temperature and humidity was conducted inside and outside to achieve a variety in atmospheric settings for temperature and humidity.

**Precision:**
The manufacturer lists the precision of the DHT11 to be ±1°C and ±1%; however, tests show an average of ±3°C and ±2%, which is a 200% and 100% error, respectively.

**Accuracy:**
The manufacturer lists the accuracy of the DHT11 to be ±2°C and ±5%; however, tests show an average of ±3°C and ±4%, which is an 50% error for temperature. The ground truth was found via various thermometers and a humidity sensor in a dehumidifier.

**Range:**
The manufacturer lists the range of the DHT11 to be 0-50°C and 20-90%. Testing both inside and outside, a relatively decent range was able to be acquired.
